# Calories to Exercise

https://calorietoexercise.vercel.app/

Calories to Exercise is a web app that converts food calories into the equivalent amount of exercise needed to burn them.

You enter a food item or calorie amount, and it shows how much running, walking, or other activity would be needed.

## What it does

- Takes calorie input and converts it into exercise equivalents
- Uses a backend API to fetch and process nutrition data
- Updates results in real time on the frontend
- Responsive layout for mobile and desktop

## Tech

- React
- Node.js
- Express
- Tailwind CSS
- Vercel (frontend)
- Render (backend)

## How it works

- Frontend sends a request with the food or calorie input
- Backend handles API calls to get nutrition data
- Returns calorie info and converts it into exercise time
- Displays results instantly in the UI
